Summary :
A web-based behavioral screening application designed to help parents identify early signs of ADHD in children aged 4–12. The system implements a fuzzy logic algorithm to analyze symptom patterns based on DSM-5 criteria and generate decision-support results. This tool is intended for early screening — not clinical diagnosis.

Problem :
•	Early ADHD symptoms are often mistaken as normal childhood behavior.
•	Delayed awareness leads to late intervention and support.
•	There is a lack of accessible digital screening tools for parents.

Goals :
•	Develop a web-based ADHD screening system.
•	Translate DSM-5 behavioral criteria into measurable inputs.
•	Implement fuzzy logic for symptom-based inference.
•	Provide clear, understandable screening results.

Scope
•	Screening tool only (not a diagnostic system).
•	Target users: parents/guardians of children aged 4–12.
•	Assessment divided into Activity and Attention sections.
•	Result classification based on fuzzy inference model.

Roles & Tools :
•	System analysis & requirement definition
•	Fuzzy algorithm design & implementation
•	Web app development (PHP, HTML, CSS)
•	System testing & validation with pediatric expert
